## 💼 Featured Projects

### 💰 [E-Wallet Microservices Application]([ADD LINK])
A distributed digital wallet platform built with 4 independent Spring Boot microservices, supporting account creation, peer-to-peer transfers, and real-time notifications.

**Tech Stack:** `Java` `Spring Boot` `Apache Kafka` `Redis` `MySQL` `Spring Security`

**Key Features:**
- 4 independently deployable microservices: User, Transaction, Wallet, and Notification
- Event-driven communication using Apache Kafka for asynchronous service coordination
- Secured inter-service authentication with Spring Security and BCrypt password hashing
- Atomic balance updates with eventual consistency across distributed services

**Learning Outcome:** Deepened understanding of distributed systems design, event-driven architecture, and the trade-offs involved in maintaining data consistency across independently deployed services.

<br>

### 📈 [DSA Stock Exchange Market]([ADD LINK])
A console-based stock exchange matching engine built from scratch using custom data structures — no external libraries, just core algorithmic design.

**Tech Stack:** `Java` `Data Structures & Algorithms`

**Key Features:**
- Custom-built SkipList for O(log n) order book insertion, deletion, and search
- Price-time priority matching algorithm with FIFO order queues for limit/market orders
- SegmentTree-based trade analytics engine for O(log n) min/max/average price and volatility queries

**Learning Outcome:** Strengthened core DSA fundamentals by implementing complex data structures from first principles rather than relying on built-in libraries.

<br>

A full-stack university enrollment management portal built with Django, streamlining the application and review process for students and admins.

**Tech Stack:** `Python` `Django` `Bootstrap` `SQLite`

**Key Features:**

- Real-time application status tracking
- Role-based admin dashboard with live enrollment statistics and review workflows
- CSRF protection and custom form validation

**Learning Outcome:** Gained practical experience with Django's MVT architecture, role-based access control, and secure file-handling in a production-style web application.

### 📚 [Digital Library Management System]([ADD LINK])
A Spring Boot backend automating library operations — from book issuance to fine calculation — with a performance-focused caching layer.

**Tech Stack:** `Java` `Spring Boot` `Redis` `MySQL` `Spring Security`
